### Introduction

**CollagePro** is a professional, privacy-focused image stitching and collage tool that runs entirely in your browser. It features an infinite canvas, smart alignment tools, and a seamless workflow for creating long screenshots, mood boards, or photo collages without uploading your data to any server.

> **Note**: All operations are performed locally. Your images never leave your device.

### âœ¨ Key Features

- **Infinite Canvas**: Pan and zoom freely to organize your workspace.
- **Smart Stitching**: Automatically stitch images vertically or horizontally.
  - **Smart Match**: Automatically resizes images to match width/height during stitching.
  - **Gap Control**: Customizable spacing between stitched images.
- **Advanced Layer Management**: 
  - Drag and drop to reorder layers.
  - Batch selection and manipulation.
  - Quick alignment tools (Left, Center, Right, Top, Middle, Bottom).
- **Precision Tools**:
  - **Snap to Grid**: Automatic alignment guides for perfect positioning.
  - **Lock Aspect Ratio**: Maintain image proportions while resizing.
- **Customizable Environment**:
  - Switch between Grid or Solid Color backgrounds.
  - Dark mode interface optimized for focus.
- **History & Export**:
  - Local version history to restore previous states.
  - High-quality PNG export with preview.
- **Responsive Design**: Fully functional on both Desktop and Mobile devices.

### ğŸ›  Tech Stack

- **Core**: React 19, TypeScript
- **Styling**: Tailwind CSS
- **Animation**: Framer Motion
- **Icons**: Lucide React
- **Build Tool**: Vite
